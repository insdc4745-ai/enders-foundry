# Ender's Foundry (엔더스파운드리)

디지털 트윈 해군 워게임 AI 시뮬레이션 — 실제 해역/함대 데이터를 디지털 트윈으로 복제하고, 그 위에서 블루/레드 AI가 전술 의사결정을 실험·단련하는 비공식 연구·교육용 합성 시뮬레이션.

## 하네스: 디지털 트윈 해군 워게임

**목표:** 검증된 오픈소스/논문 패턴(PettingZoo·MAPPO self-play·LLM 지휘 계층·디지털 트윈 온톨로지) 위에서 워게임 시뮬레이션 SW를 설계·구축하고, 그 위에서 시나리오를 점진적 난이도로 운영·생성한다.

**트리거:** 디지털 트윈 워게임/해군 시뮬레이션 관련 작업(구축·설계·구현·운영·시나리오·리서치) 요청 시 `enders-foundry-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.

**도메인 안전 제약:** 실제 무기체계 성능이 아닌 공개/합성 데이터 전제. LLM 지휘 계층은 HITL 가드레일 + 행동 검증 게이트 뒤에 둔다(에스컬레이션 위험 차단).
